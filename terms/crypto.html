Криптография

Современные алгоритмы и протоколы, устойчивые к квантовым атакам (2025)

AES-GCM

Симметричный шифр с аутентификацией (AEAD). Стандарт для TLS 1.3 и дискового шифрования.

Симметричное

ChaCha20-Poly1305

Быстрый потоковый AEAD-шифр. Используется в WireGuard, TLS, мобильных устройствах.

Симметричное

RSA (устаревает)

2048-бит всё ещё в ходу, но уже не рекомендуется. Заменяется на ECC и постквантовые алгоритмы.

Асимметричное

ECDSA (P-256, P-384)

Эллиптическая криптография для подписей. Стандарт в TLS и блокчейнах.

Подпись

Ed25519 / Ed448

Современные высокоскоростные подписи на эллиптических кривых (Curve25519). Используется в SSH, Signal.

Подпись

X25519

Diffie-Hellman на кривой Curve25519. Основной обмен ключами в TLS 1.3.

Обмен ключами

Kyber (CRYSTALS-Kyber)

Постквантовая KEM-победитель NIST 2022–2024. Уже в TLS 1.3 hybrid и Chrome.

Постквантовая

Dilithium (CRYSTALS-Dilithium)

Постквантовая цифровая подпись на решётках. Стандартизирована NIST 2024.

Постквантовая

Falcon

Альтернативная постквантовая подпись (меньше размер). Выбор для встроенных систем.

Постквантовая

SPHINCS+

Бесключевой (hash-based) постквантовый алгоритм подписи. Очень медленный, но сверхнадёжный.

Постквантовая

TLS 1.3

Обязательный 1-RTT, PFS по умолчанию, только современные шифры (нет RSA, нет SHA-1).

Протокол

Perfect Forward Secrecy (PFS)

Компрометация долгосрочного ключа не раскрывает прошлые сессии.

Свойство

Passkeys (FIDO2 + WebAuthn)

Синхронизируемые криптоключи на основе асимметричной криптографии вместо паролей.

Беспарольная

ZK-SNARKs

Доказательства с нулевым разглашением. Используется в Zcash, Ethereum L2, приватных транзакциях.

ZK

ZK-STARKs

Прозрачные ZK-доказательства без trusted setup. Масштабируемость для блокчейнов.

ZK

Fully Homomorphic Encryption (FHE)

Вычисления на зашифрованных данных без расшифровки. Прорыв 2024–2025.

FHE

SHA-256 / SHA-3 (Keccak)

SHA-256 — всё ещё основной, SHA-3 — альтернатива на губке.

Хэширование

BLAKE3

Самый быстрый криптографический хэш 2025 года (в Rust, Go, Linux kernel).

Хэширование

HMAC

Аутентификация сообщений на основе хэша. HMAC-SHA256 — стандарт.

MAC

Argon2

Современный алгоритм хэширования паролей (победитель PHC). Argon2id — рекомендация OWASP.

Пароли

scrypt

Предшественник Argon2, всё ещё используется в некоторых криптокошельках.

Пароли

Quantum Key Distribution (QKD)

Квантовая раздача ключей. Реальные проекты в Китае, Европе (2025).

Квантовая

Noise Protocol Framework

Современный фреймворк для защищённых каналов (WireGuard, Signal, Lightning).

Протокол

Signal Protocol

End-to-end шифрование + PFS + deniability. Стандарт мессенджеров.

E2EE

PGP / OpenPGP

Гибридная криптосистема для email и файлов. Всё ещё жива в 2025.

E2EE

Key Stretching

Увеличение сложности подбора пароля (PBKDF2, Argon2, bcrypt).

Пароли

Certificate Transparency

Публичный лог всех выданных TLS-сертификатов для предотвращения подделок.

PKI

Hybrid Post-Quantum TLS

X25519 + Kyber — переходный стандарт 2025 года (Chrome, Cloudflare).

Гибридная

Threshold Cryptography

Разделение ключа на части (MPC). Используется в кошельках и HSM.

MPC
