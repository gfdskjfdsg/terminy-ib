Типы атак

Самые актуальные угрозы и техники атакующих в 2025 году

MITRE ATT&CK

Глобальная матрица тактик и техник реальных атакующих групп.

Фреймворк

APT (Advanced Persistent Threat)

Долгосрочная скрытная атака с высоким уровнем мастерства.

Угроза

AI-Powered Phishing / Deepfake Vishing

Фишинг и голосовые звонки с использованием ИИ и дипфейков.

Социальная инженерия

LLM Prompt Injection

Внедрение вредоносных инструкций в промпты ChatGPT, Claude и др.

AI-атака

Token Theft / Session Hijacking

Кража OAuth-токенов, cookies, refresh-токенов из браузера или облака.

Идентичность

Living off the Land (LotL)

Использование легитимных инструментов (PowerShell, curl, certutil).

Техника

Ransomware-as-a-Service (RaaS)

Модель «вымогательство как услуга»: LockBit, BlackCat, ALPHV.

Вымогательство

Supply Chain Attack

Компрометация через сторонних поставщиков (SolarWinds, Log4Shell, 3CX).

Цепочка поставок

Cloud-Native Attacks

Атаки на Kubernetes, IAM-роли, serverless, metadata service.

Облако

Zero-Day Exploit

Использование уязвимости до выхода патча (например, MOVEit 2023–2025).

Эксплойт

Fileless Malware

Атака полностью в памяти без записи на диск.

Техника

Pass-the-Hash / Pass-the-Ticket

Использование украденных NTLM-хэшей или Kerberos-билетов.

Латеральное движение

Golden Ticket Attack

Подделка TGT в Kerberos после компрометации KRBTGT.

Active Directory

SMB Relay / NTLM Relay

Перехват и ретрансляция NTLM-авторизации.

Сеть

Watering Hole

Заражение популярного сайта, который посещают целевые жертвы.

Тактика

Island Hopping

Переход от одной жертвы к её партнёрам и клиентам.

Тактика

DDoS / Ransom DDoS

Атака отказа в обслуживании + требование выкупа.

Доступность

Memory-Safe Exploits (Rust/Go)

Новые техники обхода защит в языках без memory corruption.

2025 тренд

Initial Access Broker (IAB)

Продажа начального доступа в даркнете (RDP, VPN, Citrix).

Киберпреступность

Business Email Compromise (BEC)

Компрометация корпоративной почты для мошенничества.

Социальная инженерия

Credential Stuffing

Автоматический перебор украденных логинов/паролей.

Брутфорс

Adversarial AI / Model Poisoning

Отравление обучающих данных нейросетей.

AI-атака

Shadow AI

Несанкционированное использование сотрудниками ChatGPT и аналогов.

Риск

Malwareless Attack

Атака без вредоносного ПО — только легитимные инструменты.

Техника

Evasion Techniques

Обход EDR/ML-антивирусов (обфускация, полиморфизм).

Антивирус

Double Extortion

Шифрование + угроза публикации данных (типичная тактика 2024–2025).

Ransomware

API Abuse

Массовые запросы, обход rate limit, эксплуатация логики API.

API

IoT Botnet

Заражение умных устройств для DDoS и майнинга.

IoT

Physical Breach + Evil Maid

Физический доступ к устройству (подключение Keylogger, прошивка).

Физика

Typosquatting / Combo-squatting

Регистрация доменов, похожих на легитимные (g00gle.com).
