IDOR (Insecure Direct Object Reference)

Доступ к объектам по ID без проверки прав (user_id=123 → 124).

Логика

SSRF (Server-Side Request Forgery)

Принуждение сервера делать запросы по URL атакующего, включая внутренние сети.

Критичная

RCE (Remote Code Execution)

Выполнение произвольного кода на сервере (через deserialization, eval, include).

Максимальная

LFI / RFI

Локальное и удалённое подключение файлов через уязвимый include/require.

Включение

Path Traversal / Directory Traversal

Доступ к произвольным файлам через ../../ в параметрах.

Файловая система

Open Redirect

Неконтролируемый редирект на произвольный URL (фишинг, OAuth-обход).

Редирект

Clickjacking

Скрытие кликабельных элементов под прозрачным слоем (UI Redress Attack).

UI

CORS Misconfiguration

Неправильные Access-Control-Allow-Origin, позволяющие кражу данных.

Браузер

Шаблоны
